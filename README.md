# TY.MON SAYS

![Alt Text](./assets/game-play.gif)

## What is it?

Ty.Mon Says is a 'copy the sequence' type of game, just like SIMON. The twist is, you have a full keyboard to memorize, instead of just a few buttons. Good luck!
